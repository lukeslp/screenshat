# Screenshat

A powerful Playwright-based screenshot capture tool with 18 presets for social, mobile, and high-resolution outputs up to 16K. Features bulk URL mode, PNG tEXt alt-text embedding, and a full-stack TypeScript/Express/React implementation.

## Features

- **18 Presets**: Capture screenshots for social media, mobile devices, and high-res displays (up to 16K).
- **Bulk URL Mode**: Process multiple URLs in a single run for efficient batch operations.
- **PNG tEXt Embedding**: Automatically embed alt-text metadata in PNG outputs.
- **Full-Stack Integration**: Built with TypeScript, Express (backend), and React (frontend).
- **Public REST API**: Programmatically access screenshot capabilities (hosted version only).

**Note**: Advanced features like Claude vision analysis and hosted REST API access are available through our premium service and are not part of this open-source release.
